# Leguan's Blog 设计方案

## 概述

基于 [Kyle's Blog](https://cyborg2077.github.io/) 为参考模板，搭建一个功能完整的个人博客。

## 基本信息

| 项目 | 内容 |
|------|------|
| 博客名称 | Leguan's Blog |
| 博客标语 | Digest your emotions |
| 个人简介 | A person who wants to see the world |
| 博主名称 | Leguan |

## 技术架构

- **框架**: Hexo 7.x
- **主题**: Butterfly 4.x
- **部署**: GitHub Pages
- **包管理**: npm

## 目录结构

```
d:\blog\
├── source/
│   ├── _posts/          # 文章
│   ├── about/           # 关于我
│   ├── link/            # 友人帐
│   ├── tags/            # 标签页
│   ├── categories/      # 分类页
│   ├── archives/        # 归档页
│   ├── messageboard/    # 留言板
│   ├── shuoshuo/        # 闲言碎语
│   ├── fitness/         # 健身日寄
│   ├── bangumi/         # 番剧
│   └── memory/          # 记忆胶囊
├── themes/butterfly/    # 主题
├── _config.yml          # Hexo 配置
└── _config.butterfly.yml # 主题配置
```

## 侧边栏组件

- 博主信息卡片（头像 + 名称 + 签名）
- 文章统计（文章数、标签数、分类数）
- 公告栏
- 最新文章
- 标签云
- 友链推荐

## 页脚

- 版权信息：©2025 By Leguan
- 站点统计

## 主题样式

- 主色调：蓝紫渐变
- 支持昼夜模式切换
- 圆角卡片式布局
- 平滑过渡动画

## 启用功能

| 功能 | 说明 |
|------|------|
| 本地搜索 | 支持文章标题、内容搜索 |
| 代码高亮 | 支持多种编程语言 |
| 图片灯箱 | 点击图片放大查看 |
| 复制提示 | 复制内容时显示版权提示 |
| 页面加载动画 | 加载时显示进度条 |
| 阅读进度条 | 顶部显示阅读进度 |
| 右键菜单 | 自定义右键菜单 |

## 暂不启用功能

- 评论系统（需注册 Valine/Twikoo 等）
- 访问统计（需注册百度统计/不蒜子）
- 在线聊天

## 部署

- 平台：GitHub Pages
- 域名：leguan.github.io（或自定义域名）
